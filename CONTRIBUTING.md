# Contributing to rules-quarkus-skills

We welcome new skills related to Quarkus and Bazel!

## How to add a new skill

1.  Create a new directory under `.agent-skills/<skill-name>/`.
2.  Add a `SKILL.md` with full technical mandates and YAML frontmatter.
3.  (Optional but recommended) Add a `SKILL.toon` file for token-optimized consumption.
4.  Update the root `README.md` and `GEMINI.md` to list the new skill.

## Coding Standards

- **Mandates First:** Every skill must have a "Core Mandates" or "Architectural Mandates" section.
- **Language:** Technical content must be in English.
- **Obsidian Style:** Use callouts (`> [!TIP]`) and Mermaid diagrams for complex architectures.
