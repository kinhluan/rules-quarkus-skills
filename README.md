# rules-quarkus-skills

A collection of expert AI skills for building and maintaining Quarkus applications with Bazel.

## Available Skills

- **[rules-quarkus](./skills/rules-quarkus/SKILL.md)**: Expert mandates for maintaining the rules_quarkus project.
- **[quarkus-expert](./skills/quarkus-expert/SKILL.md)**: General Quarkus framework knowledge, reactive patterns, and CDI.
- **[bazel-expert](./skills/bazel-expert/SKILL.md)**: Starlark rules development, hermeticity, and performance optimization.

## Usage

Activate a specific skill using the Gemini CLI:
`activate_skill skills/rules-quarkus`
